Trapezoidal Rule

Suppose we need to approximate the integral \[\displaystyle \int_a^b f(x)\, dx\] using the Trapezoidal Rule with \(n\) subdivisions. Then: \[\displaystyle \int_a^b f(x)\, dx \approx T_n = \frac{h}{2} \Big[ f(x_0) + f(x_n) + 2 \Big( f(x_1) + f(x_2) + \ldots + f(x_{n-1}) \Big) \Big]\] where: \[h = \frac{b-a}{n}, \quad x_i = a + ih,\quad \text{for } i = 0, 1, 2, \ldots, n\] Also, \[x_0 = a, \quad x_n = b\]

Simpson’s Rule

Suppose we need to approximate the integral \[\displaystyle \int_a^b f(x)\, dx\] using Simpson’s Rule with \(n\) subdivisions (note: \(n\) must be even). Then: \[\displaystyle \int_a^b f(x)\, dx \approx S_n = \frac{h}{3} \Big[ f(x_0) + f(x_n) + 4 \Big( f(x_1) + f(x_3) + \ldots + f(x_{n-1}) \Big) + 2 \Big( f(x_2) + f(x_4) + \ldots + f(x_{n-2}) \Big) \Big]\] where: \[h = \frac{b-a}{n}, \quad x_i = a + ih,\quad \text{for } i = 0, 1, 2, \ldots, n\] Also, \[x_0 = a, \quad x_n = b\]
